# Customer Segmentation & Business Performance Analysis

Kicktern Internship – Task 3

## Project Objective
This project performs customer segmentation using transaction data and analyzes business performance using data-driven techniques.

The goal is to identify patterns in customer purchasing behavior and provide insights for marketing strategies and revenue growth.

## Dataset
The dataset contains:
- 6000 transaction records
- 994 unique customers
- Features: InvoiceNo, CustomerID, InvoiceDate, Quantity, UnitPrice, Country

## Methodology
The project uses the RFM Model:

Recency – How recently a customer purchased  
Frequency – Number of purchases  
Monetary – Total spending amount

Customers are segmented into:

- High Value Customers
- Loyal Customers
- At Risk Customers
- Low Engagement Customers

## Tools & Technologies
Python  
Pandas  
Matplotlib  
Jupyter Notebook  

## Key Insights
- Loyal customers contribute the highest total revenue.
- High value customers have the highest spending per customer.
- At risk customers require retention strategies.
